# Closeout Checklists Index

Load one checklist based on whether publishing is being considered.

| File | Purpose | When to use | Avoid loading unless needed |
| --- | --- | --- | --- |
| `standard-closeout.md` | End with clean state, validation, and next action. | Normal task completion or pause. | Skip if preparing to push. |
| `pre-push-closeout.md` | Verify readiness before publishing. | Before an owner-approved push. | Skip when no push is planned. |

Closeout should report dirty state, failed validation, and deferred decisions plainly.
